HOW IT WORKS

The round trip is the product.

01 — ACT

Agent does the thing.

Your agent attempts a write against the repo. No prompt, no waiting.

status: pending → daemon
02 — CAPTURE

Daemon snapshots state.

The local daemon captures the diff, the context, and a recovery snapshot — before execution.

status: snapshot → cloud
03 — REWIND

Roll back any action.

Browse the journal from the cloud. Undo anything, any time.

status: ready → reversible

↳ Escalates to a human only when an action can't be safely reversed.

PRODUCT

Three primitives. One mental model.

Capture

Snapshot state, diff, and recovery context for every action — before it runs.

Rewind

Roll back any captured action from the cloud. One click, full state.

Escalate

Stop only when reversal can't be guaranteed. A footnote, not the loop.
