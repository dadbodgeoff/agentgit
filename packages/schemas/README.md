# @agentgit/schemas

Schema pack and TypeScript contract helpers for the `agentgit` authority runtime.

## Install

```bash
npm install @agentgit/schemas
```

## Compatibility

- Node.js `24.14.0+`
- schema pack `v1`

This package exports the canonical request/response, action, run-event, timeline, and recovery types used by the daemon, CLI, and SDKs.
